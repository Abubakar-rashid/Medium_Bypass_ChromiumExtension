## Setup
---
#### 1) clone the repository into any folder
#### 2) Open you browser settings
#### 3) Go to extension section and turn on the developer mode 
#### 4) Click load un-packed and select the folder you have copied the repo in
### Greate !! You have successfully setup the extension, now you can go to any medium article and at the bottom right you will see a go to freedium button, upon clicking you will be redirected to page with full unlocked article.
